PERO.. COMO FUNCIONA UNMOTOR

Un motor de auto es como un corazón que hace que el auto se mueva. Funciona quemando gasolina para hacer girar pistones. Estos pistones empujan hacia abajo y hacen girar cigueñal. El cigüeñal hace girar las ruedas, ¡y eso hace que el auto se mueva!

Admisión: Es como tomar una gran bocanada de aire. El motor abre una válvula para dejar entrar aire y gasolina dentro del pistón. Es como si estuviera respirando para prepararse. Compresión: Luego, el pistón aprieta esa mezcla de aire y gasolina. Es como cuando aprietas un resorte. Esto hace que la mezcla se vuelva más fuerte y lista para explotar. Explosión: Ahora, el motor hace "boom". La mezcla de aire y gasolina explota con una chispa de la bujía. Es como si encendieras un fuego dentro del motor. Expulsión: Finalmente, el motor empuja hacia afuera los gases que quedaron después de la explosión. Es como soplar después de apagar una vela. Así, el motor se prepara para empezar todo de nuevo.
